weatherService

Input: String zipcode

Output: 2d list of strings consisting of 5 day weather forecast information

TryIt

Service returns the 5-day weather forecast for a given zipcode. The out put is 2d list of strings consisting of 6 rows and 5 columns. Row-1  shows format- ”Date", "minTemperature", "maxTemperature", "Day conditions", "Night conditions". Rows 2-6 , hold corresponding values for each of the 5 days

Use apis provided by accuWeather.com. Two of their Apis are required. First, Postal Code Search (results narrowed by countryCode) Api is  to get Location Key. Second, 5 Days of Daily Forecasts api to get the forcasts

AirQuality

Input: Decimal Lattitude , Decimal Longitude

Output: String of format “{Air quality Index},{Pollution Level}”

TryIt

This service returns the airquality index and Pollution Level status for a given lattitude and longitude. Return string has airaquality index followed by a comma and the pollution level status.

Use api is provided by http://aqicn.org/api/.

Use their gelocalized feed (lat/lng based) to get airquality index. Use Information provide in http://aqicn.org/faq/ to compute pollution level.

Use apis provided by yelp.com to get the necessary information. Use Businesses Search api to get the nearest businesses information.
